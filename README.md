# vile-eslint

A [vile](http://vile.io) plugin for [eslint](http://eslint.org).

## Requirements

- [nodejs](http://nodejs.org)
- [npm](http://npmjs.org)

## Installation

    npm i vile-eslint

## Config

```yaml
eslint:
  config: .eslintrc
  ignore: .eslintignore
```

## Hacking

    cd vile-eslint
    npm install
    npm test
